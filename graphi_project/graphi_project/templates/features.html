Features

Directed Graph

Graphi uses Javascript to draw a directed graph of function calls. Each node represents a function,
and an edge represents a call from a function to another function.

A node's label in the graph has the layout: function-name total-time-% (self-time-%) total-calls.

Function name has the format: file:line-number:function-name.
Total time % is the % of running time spent in the function and its children.
Self time % is the % of running time spent in the function itself.
Total calls is the total number of times the function was called.

Sortable Table

Graphi also creates a sortable table of function calls. The table's labels have the same meaning as
the labels in the graph.

Click on a table header to sort the table by that header. The table can also
be adjusted to display more rows or searched.
