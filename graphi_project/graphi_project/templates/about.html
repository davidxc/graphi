About

Graphi is a web application for drawing directed graphs from the output of profilers.
Graphi also creates a sortable table of function calls and running times.

Graphi currently only supports output from Python's cProfile. Support will be extended
to other profilers in the future, probably starting with profilers for Ruby and C.

The code for Graphi is also on Github if you want to run it locally. To report bugs
or request features, please email the developer at davidwong.xc@gmail.com .
